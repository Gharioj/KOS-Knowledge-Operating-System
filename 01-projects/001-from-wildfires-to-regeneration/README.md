# From Wildfires to Regeneration

## Summary

Project 001 develops the From Wildfires to Regeneration Framework — moving from wildfire crisis toward long-term ecological and social regeneration. This folder holds the Framework's Version 1.0 publication suite: its founding manuscript, its capability structure, and the materials prepared for governance, patron, and strategic-partner audiences. Version 1.0 is approved for external distribution.

## Version 1.0 Documents

| Document | Description |
|---|---|
| Dagli Incendi alla Rigenerazione (Founding Manuscript) | The original conceptual foundation of the Framework, retained as a permanent historical record. |
| From Wildfires to Regeneration — Executive Board Paper | Presents the Framework to a governing board. |
| Framework Capability Register | Documents the Framework's capability structure for Version 1.0. |
| Patron Invitation | Invites patrons to support the Framework. |
| Strategic Partner Proposal — Marcegaglia | Proposes a strategic partnership with Marcegaglia. |

Full file names, status, and language are recorded in [04-publication/README.md](04-publication/README.md).
