# Archive

Superseded or retired material, kept rather than deleted, per [00-system/ai/PRINCIPLES.md](../00-system/ai/PRINCIPLES.md).

## Convention
Archiving moves an item here, preserving its depth in the tree — a project at `01-projects/NNN-slug/` becomes `06-archive/NNN-slug/`. Because depth from the repository root is unchanged, internal relative links keep resolving. Set `status: archived` before moving it.
