# Knowledge

Evergreen knowledge that outlives any single project: concepts, frameworks, and shared terminology. If it would still be true after the project that produced it is archived, it belongs here, not inside `01-projects/`.

- [glossary.md](glossary.md) — controlled vocabulary; tags used anywhere in the system must appear here first
- `concepts/` — standalone ideas and definitions
- `frameworks/` — reusable models and methods

New entries use [00-system/templates/document.md](../00-system/templates/document.md).
