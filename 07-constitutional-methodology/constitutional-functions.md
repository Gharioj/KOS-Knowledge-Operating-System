# Constitutional Functions

## Definition

A Constitutional Function is a recurring, nameable pattern of work that a system depends on being done — the point at which a [Human Capacity](human-capacities.md), exercised repeatedly and predictably, becomes something that can be organised around, staffed and held accountable. A function is not a person, a role, or a department; it is the stable shape that demand for a capacity takes once it recurs often enough to be named.

## Constitutional Principle

A function has no standing of its own. It exists only as long as it channels a real human capacity toward a need the system actually has. Naming a function does not create the capacity behind it — it only makes an existing pattern visible enough to organise. A function that persists after the capacity or the need behind it has gone is a vestige, not a constitutional function, and should be retired rather than defended.

## Why Constitutional Functions Are the Link Between Capacity and Institution

Capacities are individual and particular — this person's ear, that person's negotiating skill. Institutions are collective and durable — a company, a label, a partnership that outlasts any one person. Between the two sits the function: the abstraction that lets a particular capacity be recognised as fulfilling a general, recurring need, so that it can be organised, staffed, trained for, or replaced without the underlying need going unmet. Without functions, institutions would have no way to organise capacity beyond individual relationships; without capacities, functions would have nothing to organise.

## Relationship to Human Capacities

A function cannot be defined independently of the [Human Capacity](human-capacities.md) that performs it. It is only ever a name for a capacity in recurring use — the pattern that emerges once enough people exercise a similar capacity toward a similar need for the demand to be worth organising.

## Relationship to Constitutional Institutions

[Constitutional Institutions](constitutional-institutions.md) organise functions over time, giving them continuity, resourcing and lines of accountability that no single instance of a function could sustain alone. An institution is, structurally, the set of functions it has taken responsibility for — its legitimacy rests on performing them faithfully, not on its formal existence.

## Examples from the Creative Ecosystem

- **A&R** — the function of locating and validating talent, performed by scouts, executives or labels acting on the same underlying capacity.
- **Promotion** — the function of building audience awareness, performed by pluggers, publicists or marketers.
- **Financial administration** — the function of royalty accounting and licensing, performed by business managers or administrators.

Each names a recurring pattern of demand, independent of who currently performs it or which institution currently houses it.

## Relationship to Technology

Technology can absorb, reshape or accelerate how a function is performed — automating routine parts of A&R screening, royalty calculation or audience analysis — without eliminating the human capacity the function ultimately depends on. Technology changes how much capacity a function requires and who is positioned to supply it; it does not remove the underlying need that gives the function its constitutional standing.

## Notes

Keep function definitions generic to the pattern of work, not to any current tool, company or job title — those change; the underlying pattern of demand is what stays evergreen.
