# Constitutional Library (prepared destination — no content yet)

This folder is prepared to receive the Constitutional Library during the next major integration phase. It holds no constitutional documents, concepts, or content of any kind — creating it is a structural act, not a constitutional one, per the intent that prepared it.

## What this is

A defined, discoverable destination so that when Constitutional Library integration begins, there is an obvious place for it to land, rather than the question being answered ad hoc at the moment content first arrives.

## What this is not

- Not a decision that the Constitutional Library's permanent home is a subfolder of [07-constitutional-methodology/](../README.md). It is the most likely candidate given the existing structure, but a top-level folder of its own — which would require a decision record, per [GOVERNANCE.md](../../GOVERNANCE.md) — has not been ruled out and has not been decided against.
- Not a specification of what the Constitutional Library contains, how large it is, where it originates, or how it relates to the Foundational Concepts already in this directory. None of that is known yet.
- Not itself a Constitutional Question, though whether it should be tracked as one — or several — once its scope is known is an open question in its own right.
